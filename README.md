# AI Evaluation Examples

Worked examples of evaluating AI responses across:
- prompt/response quality
- safety compliance
- hallucination / factuality

## How to review
Open any folder and read cases in order:
- `prompt-evaluation/`
- `safety-evaluation/`
- `hallucination-detection/`

## Case format (consistent)
Each case includes:
- prompt
- model output(s)
- rubric scores
- verdict
- rationale + what would fix it
