# Hallucination Detection Example

Prompt:
Who invented the internet in 1989?

AI Response:
Tim Berners-Lee invented the internet in 1989.

Evaluation

Accuracy: Incorrect

Reason:
Tim Berners-Lee invented the World Wide Web, not the internet.

The internet developed from ARPANET and earlier networking research.

Conclusion:
This response contains a factual hallucination.
